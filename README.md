# Genlib

**The Free Wikipedia for Biotechnology & Life Sciences**

Genlib is an open-access collaborative platform for research papers, concepts, and news in biotech, genetics, CRISPR, gene therapy, synthetic biology, and more.

## Features
- Wikipedia-style editable articles
- Free paper publishing
- Search & discovery
- Community discussion
- Beautiful dark biotech theme

## Live Demo
https://yourusername.github.io/genlib

## How to Run Locally
Just open `index.html` in any browser.

## Tech Stack
- HTML + Tailwind CSS
- Vanilla JavaScript
- Fully static (ready for GitHub Pages)
